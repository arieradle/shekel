## Registering with Google Search Console

### Step 1: Access Google Search Console

1. Go to [Google Search Console](https://search.google.com/search-console)
2. Sign in with your Google account
3. Click "Add Property"

### Step 2: Choose Verification Method

You have several options:

#### Option A: Domain Verification (Recommended)

1. Select "Domain" property type
2. Enter: `arieradle.github.io`
3. Verify domain ownership via DNS TXT record
4. Add the TXT record to your DNS settings

#### Option B: URL Prefix Verification (Easier for GitHub Pages)

1. Select "URL prefix" property type
2. Enter: `https://arieradle.github.io/shekel/`
3. Choose verification method:

**HTML Tag Method (Easiest):**

1. Google will provide a meta tag like:
   ```html
   <meta name="google-site-verification" content="YOUR_CODE_HERE">
   ```

2. Add this to `docs/overrides/main.html`:
   ```html
   <!-- Find this line in the file -->
   <!-- <meta name="google-site-verification" content="YOUR_VERIFICATION_CODE_HERE"> -->
   
   <!-- Replace with your actual code -->
   <meta name="google-site-verification" content="YOUR_ACTUAL_CODE">
   ```

3. Commit and push:
   ```bash
   git add docs/overrides/main.html
   git commit -m "Add Google Search Console verification"
   git push origin main
   ```

4. Wait for GitHub Actions to deploy (~2 minutes)
5. Return to Google Search Console and click "Verify"

**HTML File Method:**

1. Google will provide an HTML file to download
2. Place it in the `docs/` directory
3. Commit and push
4. Click "Verify" in Google Search Console

### Step 3: Submit Sitemap

Once verified:

1. In Google Search Console, go to "Sitemaps" (left sidebar)
2. Enter sitemap URL:
   ```
   https://arieradle.github.io/shekel/sitemap.xml
   ```
3. Click "Submit"

The sitemap is automatically generated by MkDocs and includes all pages.

### Step 4: Monitor Indexing

After submission:

1. **Coverage Report**: Check which pages are indexed
2. **URL Inspection**: Test individual page indexing
3. **Performance**: Monitor search performance
4. **Search Appearance**: See how your site appears in results

## Troubleshooting

### Verification Failed

- **Problem**: HTML tag not found
- **Solution**: 
  1. Check that you uncommented the line in `docs/overrides/main.html`
  2. Ensure you pushed and the site deployed
  3. Clear your browser cache
  4. Try the HTML file method instead

### Sitemap Not Found

- **Problem**: `sitemap.xml` returns 404
- **Solution**: 
  1. MkDocs generates this automatically
  2. Run `mkdocs build` locally to verify
  3. Check that the file exists in the `site/` directory
  4. Ensure GitHub Actions deployment succeeded

### Pages Not Indexed

- **Problem**: Pages aren't appearing in search
- **Solution**:
  1. Wait at least 1 week after submission
  2. Check Coverage Report for errors
  3. Use URL Inspection tool
  4. Manually request indexing for important pages
